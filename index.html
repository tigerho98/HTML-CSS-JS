    1. title 속성 : 링크 위로 마우스를 옮겼을 때 나타날 텍스트 표시
      < a href="https://www.naver.com" title="네이버로 가볼까?">네이버 < /a>
      네이버

    2. target 속성 :
      • ="_blank" : 새 탭에서
      • (default) : 현재 탭에서
    3. path(directory) 상위 디렉토리의 폴더 : ../

  • em, i : 이탤릭체 strong, b : 굵게 u : 밑줄
    this is em tag

    • 컨텐츠 구조화

    • header : 큰 제목과 로고, 주요 정보
      • < header >

    • navigation bar : 홈 페이지의 메인 섹션; 메뉴 버튼이나 링크, 탭으로 표현; 다른 페이지로 넘어가도 구성으로 남아있음
      • < nav >

    • main content : 중심 컨텐츠
      • < main > < article > < section > < div >

    • sidebar : 주변의 정보, 링크, 인용부호, 광고
      • < aside > 주로 main안에 위치

    • footer : 작은 정보, 저작권 정보, 연락처 등 부차적인 정보
      • < footer >
